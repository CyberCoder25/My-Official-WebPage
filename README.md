My Official WebPage

Project Overview
- My Official WebPage is my first full-stack project
- A professional portfolio website showcasing personal projects, skills, services, and a secure contact interface
- Combines a Node.js backend with a responsive frontend built using HTML, CSS, and JavaScript

Key Highlights
- Full-stack integration of frontend and backend
- Secure coding practices
  - CSRF protection on all forms
  - Input validation and sanitization
  - Safe handling of environment variables
- SEO optimized pages for search engines
- Fully responsive and accessible design

Project Structure

Installation and Setup
- Clone repository
  - git clone https://github.com/CyberCoder25/My-Official-WebPage.git
  - cd My-Official-WebPage
- Setup backend
  - cd backend
  - npm install
  - npm start
- Open frontend
  - Open frontend/index.html in a browser
  - Or serve using any static server
